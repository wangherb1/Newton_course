# Decision: 固化样板页主体四模块基线

日期：2026-06-01

## 背景

命题1定理1样板页已经完成多轮从上到下人工检查。Hero 首端封面、英文原文、中译版和现代微积分证明四个主体模块已经达到可复用水平。动态图、思考题等扩展模块仍需继续审阅。

## 决策

1. 将四个主体模块固化为后续节点建设基线。
2. 新增 `docs/standards/sample_page_body_baseline.md` 统一记录结构、来源和视觉规则。
3. 动态图、思考题、命题网络、评论区和署名模块继续保留，但标记为待审阅。
4. 不把当前扩展模块形态提前复制为最终范式。
5. 使用 `archive/2026-06-01_prop001_theorem001_body_baseline/` 保存阶段快照、关键文件哈希和下一线程续接摘要。

## 影响

- 后续页面可直接复用主体四模块结构，减少重复讨论。
- 扩展模块仍可独立迭代，不阻塞主体内容扩展。
- 历史状态可通过留档清单和 SHA-256 校验值追溯。
