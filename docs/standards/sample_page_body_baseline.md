# 样板页主体四模块基线

## 适用范围

本规范以 `principia-book1-chapter02-prop001-theorem001` 为首个样板页，固化已经过人工复核并基本敲定的主体正文范式。后续定义、定律、引理、命题、定理、推论和附注页面应优先复用本基线。

样板页开发地址：

```text
http://127.0.0.1:4321/nodes/book1-chapter02/prop001-theorem001/
```

## 已确认主体模块

### 1. Hero 首端封面

- 使用 AI 单层背景图，不叠加第二套 SVG、公式或精确符号。
- 左侧保留标题空间，右侧提供与节点主题一致的弱语义视觉。
- 精确科学图示进入正文模块。
- 命题1定理1当前固定使用 `prop001_theorem001_hero_ai_v1.png`。

### 2. Andrew Motte 1729 英文原文

- 日常数字主源使用 Wikisource 1846 页面；译本归属仍写 Andrew Motte 1729 英译本。
- 命题或定理核心句独立加粗斜体突出。
- 原版插图按原文相对位置插入。
- 原页面没有紧邻图题时，不额外增加图号、图注或单图链接。
- 来源行保留页面级文本，后续统一规范化前不得丢失来源定位。

### 视频课模块

- 视频课模块置于英文原文模块之前。
- 后续每个命题、定理、推论页如有老师线下讲课视频，优先在此模块接入。
- 视频资产存放在 `assets/videos/<chapter>/<node>/`，并通过 `scripts/sync_web_public_assets.py` 同步到 `web/public/assets/videos/<chapter>/<node>/`。
- 页面端只引用 `web/public` 下的发布路径，例如 `/assets/videos/book1_chapter02/prop001_theorem001/feynman_equal_area_example.mp4`。
- 样板页当前使用费曼相关讲解视频作为模块演示占位，正式发布前应替换为授权确认后的课程视频。

### 3. 中译版

- 标题使用“中译版”，英文小标题使用 `Chinese translation`。
- 页面内不重复显示稿件来源说明；来源信息保留在节点元数据和内部审阅状态中。
- 命题或定理核心句独立加粗斜体突出。
- 老师 Word 稿件插图按相对位置插入；如原图包含无关界面，保留溯源原图并另生成网页裁剪版。
- 桌面端插图右浮动并允许正文绕排；窄屏改为整行显示。

### 4. 现代微积分证明

- 标题使用“现代微积分证明：极坐标方法”，英文小标题使用 `Modern calculus proof in polar coordinates`。
- 页面内不重复显示稿件来源说明。
- 优先从老师提供的 `- latex.docx` 配套稿件录入公式，原始 Word 内嵌对象负责视觉回查。
- 公式使用 KaTeX 渲染；复杂公式必须进行语义检查、桌面截图复核和窄屏截图复核。
- 长公式在窄屏中允许公式本体横向滚动，公式编号固定留在右侧。
- 正文引用图号时，插图必须显示一致图题。命题1定理1使用“图2.1 仅受向心力作用的物体的运动”。

## 图示与动态演示质量要求

- 动态演示必须先明确节点核心展示点，不得只做普通轨迹动画或装饰性运动。
- 演示应同时提供形象视觉和可核查证据：几何结构、关键量标注、数值或状态提示必须服务于同一结论。
- 命题1定理1的核心展示点是“固定力心、焦点连线、等时间、等面积”。样板页采用椭圆轨道，把力心置于焦点，显示焦点连线长度随位置变化，同时显示各等时间段扫过的面积相等。
- 正文若提到“图示”或“动态图示”的结论，演示区的标题、标注和说明必须与正文引用保持一致。
- 桌面和窄屏都必须复核：椭圆、焦点、当前物体、面积扇形和状态面板不得互相遮挡。
- 动态演示的页面文字必须面向正式课程读者，采用客观学术叙述，不使用内部制作说明语气。
- 每个动态演示应先生成同目录 prompt 文档，再生成独立 HTML 文件；网页端应直接挂载这个 HTML 文件，保证人工修改 HTML 后网页同步呈现。
- 动态演示 HTML 的主体内容不能只做图形。应包含命题陈述、交互控制、几何与面积检验、近远关键情形对比、图例和读图要点；图框区可采用课程页视觉风格，文字面板应服务于正式读者理解。
- 桌面端动态图框区应固定，右侧文字面板独立滚动；移动端不得出现横向裁切。
- 涉及物理或数学定律时，动画取样必须与定律一致。命题1定理1的椭圆示例不得按等角速度取样，应按平近点角 `M = E - e sin E` 等间隔生成相等时间段，并用面积数值或误差校验支撑“等面积”结论。
