# 英文原版插图范式

## 适用范围

英文原文模块中的原版插图用于保留历史文本的图文关系，不替代正文讲解区的重绘图、现代示意图或互动演示。

## 来源规则

1. 日常定位使用 Wikisource 1846 数字页面。
2. 优先下载页面中对应的 Wikimedia 图片，不使用截图裁切。
3. 项目主资产存放在 `assets/images/primary_sources/`，发布副本由 `scripts/sync_web_public_assets.py` 同步到 `web/public/assets/primary-sources/`。
4. 图片必须记录来源页面、Wikimedia 文件名和本地资产路径。
5. 页面显示位置尽量接近 Wikisource 1846 页面中的原始插入位置。

## 展示规则

1. 桌面端允许右浮动，使英文正文自然绕排。
2. 窄屏改为整行显示，避免正文过窄。
3. 原版页面没有紧邻图片的图号或说明时，网页中不额外添加图注、图号或单图链接。
4. 图片来源继续记录在页面级元数据和资产清单中，不在正文插图下重复展示。
5. 原版插图与课程重绘图必须分区展示，不得混淆用途。
