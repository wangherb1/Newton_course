# 评论与互动策略

## 当前阶段

1. 不自建真实评论后端。
2. 每个命题、引理页面预留讨论区组件。
3. 讨论区显示“评论功能规划中，当前版本暂未开放”。
4. 当前组件为 `web/src/components/DiscussionPlaceholder.tsx`。

## 后续候选

- Canvas 或 MOOC 平台讨论区
- GitHub Discussions + Giscus
- Utterances
- Remark42
- Commento
- 自建审核后台

## 启用条件

未来若开放公开评论，必须支持删除、审核、关闭、置顶、防广告和防不良信息。选型前应单独记录 decision log。
