# 命题1定理1动态图示升级记录

## 时间

2026-06-03

## 范围

- 样板页“图示与动态演示”模块。
- `AreaLawDemo` React SVG 交互组件。
- 动态图示质量范式与项目状态记录。

## 核心展示点

命题1定理1要展示的是：物体仅受指向固定力心的向心力作用时，物体与力心的连线在相等时间内扫过相等面积。

本轮演示采用椭圆轨道和焦点力心，避免圆轨道特殊情形遮蔽核心结论。演示必须让用户同时看到：

- 椭圆轨道。
- 固定焦点力心 S。
- 各等时间段对应的面积扇形。
- 焦点连线长度随位置变化。
- 连线长度不同但每段扫过面积相等。

## 修改结果

- 将旧 `AreaLawDemo` 圆轨道原型替换为椭圆等面积交互演示。
- 使用等面积参数生成椭圆上的等时间点。
- 增加面积扇形、焦点连线、当前物体、切线趋势和向心力方向。
- 状态面板显示当前等时间段、每段面积、连线长度范围和本步连线长度变化。
- 页面标题改为“图示与动态演示：椭圆轨道中的等面积定律”。
- 更新动态图示质量范式。

## 待审阅

- 演示是否足够直观地表达“长度不同、面积相等”。
- 窄屏布局是否满足正式页面要求。
- 是否需要进一步增加连续动画轨迹、暂停状态标注或更严格的数学说明。
